# Firefox WebExtension _Web Theme_

Firefox WebExtension that applies a dark theme for certain websites.
 
Implemented completly:

* _none_
 
Implemented partly:

* `http*://*.google.de/*`

Planned:

* `http*://*.github.com/*`
* `http*://*.facebook.com/*`

Only CSS styles used, see [manifest.json](manifest.json).
